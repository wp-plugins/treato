Treato provides usable healthcare insights by travels around the Web to collecting and analyzinge patient experiences written in forums and blog posts. This plugin provides a snapshot of Treato search results, with links to full posts. The plugin gives access to this new aggregated information.

== Description ==

Treato travels around the Web to collect and analyze patient experiences written in forums and blog posts. Our patented semantic technology analyzes aggregates the posts in a way to generate useful insights help our users understand more about the users' medications, they are taking and the diseases and symptoms they're experiencing - all from other patients experiences. With over 1 Billion collected posts, our users receive insight about effectiveness, side effects, drug comparisons, which drugs people feel work, and  much more.
The widget's size is 300x350 pixels, it which can fits in most sidebars, and we are working on fitting it into sidebars narrower than 300 pixels, too.


Visit Treato at http://treato.com to learn more about what this plugin can provide your blog.
You can also see it live on our blog: http://treato.com/blog/

http://www.youtube.com/watch?v=8rEXhLXfOWM

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your sidebar.
